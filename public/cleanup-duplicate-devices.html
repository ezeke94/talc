This tool will remove duplicate device registrations from all users' accounts.

What it does:

  • Scans all users for duplicate device registrations
  • Identifies same physical device with multiple FCM tokens (by userAgent + platform)
  • Keeps the most recently used token for each device
  • Removes older tokens from the same device
  • Optionally removes old fcmToken field from user documents

Note: This is safe to run and won't affect active devices. It cleans up old tokens when FCM refreshes tokens.
